## 아스키 코드

문제 출처 : https://www.acmicpc.net/problem/11654

- 문제 

  알파벳 소문자, 대문자, 숫자 0-9중 하나가 주어졌을 때, 주어진 글자의 아스키 코드값을 출력하는 프로그램을 작성하시오.

  

- 입력

  알파벳 소문자, 대문자, 숫자 0-9 중 하나가 첫째 줄에 주어진다.

  

- 출력

  첫째 줄에 다음 세 가지 중 하나를 출력한다.입력으로 주어진 글자의 아스키 코드 값을 출력한다.

  
  
- 예제

  | number | 예제 입력 | 예제 출력 |
  | ------ | --------- | --------- |
  | 1      | A         | 65        |
  | 2      | C         | 67        |
  | 3      | 0         | 48        |
  | 4      | 9         | 57        |
| 5      | a         | 97        |
  | 6      | z         | 122       |
